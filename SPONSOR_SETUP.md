# 贊助頁設定 — 綠界 ECPay

## 1. 申請綠界個人賣家

1. 到 https://www.ecpay.com.tw → 註冊 → 選「個人賣家」
2. 上傳：身分證正反面 + 銀行存摺（單一本人銀行帳號）
3. 等待綠界審核（3-5 個工作日）
4. 開通「全方位金流（個人）」服務

**手續費**：信用卡 2.8% + NT$ 5／筆；ATM／超商代碼類似區間。
**對價要求**：個人賣家可賣「數位商品」，類別歸「服務／其他」。

## 2. 在綠界後台建立兩個商品

進入後台 → 商品管理 → 新增商品：

### 商品 A：贊助方案 · 一杯咖啡
- 商品名稱：贊助方案 · 一杯咖啡（感謝名單列名）
- 商品分類：服務 / 其他
- 售價：NT$ 90
- 商品說明：「贊助 ChouLegal 開發者持續維護民眾版三站免費法律工具。回饋為『感謝名單標準列名』於 https://choulegal.com/sponsor.html 公告。完成付款後 7 個工作日內公告。名字／訊息可於結帳備註欄填寫。」
- 退費政策：「名字公告後依消保法 §19 III 不再受理退費。公告前可 email 撤回全額退費。」

### 商品 B：贊助方案 · 一壺咖啡
- 商品名稱：贊助方案 · 一壺咖啡（感謝名單金色列名）
- 商品分類：服務 / 其他
- 售價：NT$ 300
- 商品說明：「贊助 ChouLegal 開發者持續維護民眾版三站免費法律工具。回饋為『感謝名單金色標記列名』於 https://choulegal.com/sponsor.html 公告。完成付款後 7 個工作日內公告。名字／訊息可於結帳備註欄填寫。」
- 退費政策：「名字公告後依消保法 §19 III 不再受理退費。公告前可 email 撤回全額退費。」

## 3. 取得「結帳連結」

每個商品建立後綠界會給一個「結帳連結」，格式類似：
```
https://payment.ecpay.com.tw/QuickCollect/...
```

複製「一杯」和「一壺」兩個連結。

## 4. 把連結貼進 sponsor.html

編 `choulegal-site/sponsor.html`，找到兩處 placeholder：

```html
<a class="tier" href="ECPAY_CUP_URL" ...>     ← 換成「一杯」結帳連結
<a class="tier featured" href="ECPAY_POT_URL" ...>  ← 換成「一壺」結帳連結
```

存檔 → `git commit + push` → Vercel 自動 deploy 上線。

## 5. 收到通知後

綠界會 email 你「訂單通知」，內含：
- 贊助者 email
- 結帳備註（如果他填了名字／訊息）
- 入帳金額

**列入感謝名單**：
1. 編 `choulegal-site/supporters.json`
2. 在 `supporters[]` 加 entry：
   ```json
   { "name": "義大利麵店王老闆", "tier": "pot", "message": "謝謝這個工具", "date": "2026-06-15" }
   ```
3. `git commit + push`

**對方沒填名字**：用 `"name": "匿名"`。

## 6. 開發票 / 稅務

個人賣家年度收入 < NT$ 200,000 可不開發票；超過要：
- 開電子發票（綠界後台可一鍵開立）
- 報綜所稅「其他所得」

第一年通常不會超過門檻，先別擔心。

## 7. 退費處理

若有人 email 要求退費：
1. 確認名字尚未公告於 sponsor 頁
2. 綠界後台 → 訂單 → 退款
3. 通常 7-14 個工作日入帳贊助者帳戶
