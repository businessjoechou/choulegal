App 隱私權政策

最後更新：2026-07-07 · 適用：ChouLegal 周全法律權益 App（iOS / Android）1.0.0 起

一句話總結

本 App 不強制登入，法律工具可免帳號使用。未登入時，你所輸入的金額、日期、人數等試算資料皆只儲存在自己的裝置上，不會離開手機。若你選擇登入「周全帳號」，本 App 會處理登入所需的電子郵件與登入狀態。

注意：本頁僅適用於本公司 iOS / Android App 版本「ChouLegal 周全法律權益」。網頁版（choulegal.com）之隱私政策請見 choulegal.com/privacy，兩者規範不同。

一、我們是誰

「ChouLegal 周全法律權益」App（下稱「本 App」）由周全科技 ChouLegal（下稱「本公司」）開發提供，提供台灣民眾免費使用之法律計算工具。

聯絡方式：businessjoechou@gmail.com（主旨註明「App 隱私」優先處理）

二、未登入使用時，我們不蒐集的資料

• 不要求註冊帳號、不要求登入、不要求 Email、不要求姓名、不要求電話。

• 不蒐集姓名、身分證、地址、聯絡方式或任何足以識別個人身分之資料。

• 不蒐集你在計算機中輸入的任何金額、日期、案件內容或法律問題。

• 不使用任何第三方分析服務（無 Google Analytics、無 Firebase Analytics、無 Facebook SDK、無廣告 SDK）。

• 不使用 Cookies、不使用裝置廣告 ID（IDFA / GAID）。

三、選擇登入周全帳號時處理的資料

若你在 App 內選擇登入或建立周全帳號，本 App 會透過 Supabase Auth 處理：

• 你輸入的電子郵件地址。

• 登入連結寄送、登入 token、session refresh token 等帳號驗證資料。

• 帳號識別碼、電子郵件、顯示名稱等周全帳號基本資料。

上述資料僅用於登入、維持登入狀態與銜接周全旗下產品服務；不會用於廣告投放或出售給第三方。

四、僅儲存於你裝置本地的資料

為提供「我的最愛」、「最近使用」功能，本 App 會在你的手機本機（透過 React Native AsyncStorage）儲存以下資料：

• 你加入收藏的計算機代號清單

• 你最近開啟的計算機代號清單

上述資料僅儲存於你的裝置，本公司無法存取，亦不會傳輸至任何伺服器。當你解除安裝本 App 時，這些資料隨之刪除。

五、系統權限

本 App 不要求任何系統權限（不要求相機、相簿、麥克風、定位、通訊錄、通知等）。

六、資料的使用、分享與傳輸

• 未登入試算資料不傳輸：你在計算工具輸入的內容不會上傳至本公司伺服器。

• 帳號資料用途限定：登入 Email 與 session 僅用於帳號驗證與維持登入狀態。

• 服務提供者：本 App 使用 Supabase Auth 處理帳號登入，該服務可能涉及境外資料處理。

• 無廣告分享：本公司不會將帳號資料提供給廣告商或資料仲介。

七、兒童隱私

本 App 不刻意針對 13 歲以下兒童設計。若未成年人使用帳號功能，應由法定代理人陪同或同意。

八、資料安全

未登入使用時，你所輸入的試算內容不會經由網路傳送。登入功能由 Supabase Auth 處理，登入 token 會透過裝置本機儲存維持 session。本機儲存資料受裝置作業系統 sandbox 機制保護。

九、你的權利

你可隨時：

• 將已收藏工具取消收藏。

• 解除安裝本 App，所有本機資料即刻刪除。

• 對周全帳號資料行使個資法第 3 條所定查詢、補充、更正、刪除等權利；請來信 businessjoechou@gmail.com。

十、法律免責聲明

本 App 提供之法律計算結果僅供參考，不構成法律意見、不取代律師、會計師、地政士等專業人員之諮詢。實際法律效果可能因個案事實、最新法令修正、判決見解而異。請於重要法律事務上諮詢合格專業人員。

十一、政策變更

本政策若有重大變更，將於本 App 內與本頁面公告。持續使用本 App 視為你接受變更後之政策。

十二、適用法律與管轄

本政策依中華民國法律解釋與適用，因本政策所生之爭議以台灣台北地方法院為第一審管轄法院。

十三、聯絡方式

如對本政策有任何疑問，請來信：businessjoechou@gmail.com
