### ① 責任制 ≠ 沒加班費 〔憤怒共鳴〕

```
老闆:「我們責任制,沒有加班費。」

但「責任制」不是老闆說了算——
全台灣只有勞動部公告的那幾種職類可以用(勞基法 §84-1),
你那個行政、客服、業助職位,八成根本不在名單上。

不在名單 = 他講的責任制 = 違法不給加班費。

平日加班前 2 小時 ×1.33、第 3 小時起 ×1.67(勞基法 §24),
追溯時效還有 5 年。

```

### ② 資遣費自己會算,公司就 A 不到你 〔實用收藏〕

```
被資遣的當下最容易被坑,因為你慌、他不慌。

資遣費公式其實超簡單(新制,2005/7 後到職):

這 6 個月加班、領獎金越多,資遣費越多。

### ③ 「自願離職書」千萬別簽 〔故事+警示〕

```
朋友被資遣,公司卻拿一張「自願離職書」要他簽,還說「這只是流程」。

差點簽下去。

被資遣 vs 自願離職,差的不是面子,是錢:
・自願離職 → 資遣費 0
・自願離職 → 失業給付也領不到(那是給「非自願離職」的)

公司很愛誘導你自己寫離職單,就是為了省這兩筆。
真要資遣,雇主應該主動給你「資遣費 + 非自願離職證明書」。

簽名前,先想清楚誰受益。
```

**法源:** 勞基法 §16、§17;就業保險失業給付

### ④ 加班費三種「日」倍率,搞混就少領 〔乾貨圖卡〕

```
台灣的「加班費」不是一個數字,是三種:

📌 平日加班(勞基法 §24)
　前 2 小時 ×1.33,第 3 小時起 ×1.67

📌 休息日上班(§24 II)
　前 2 小時 ×1.33,第 3 小時起 ×1.67,按實際時數核計

📌 例假/國定假日/特休上班(§39)
　工資「加倍發給」= 當天領 2 倍

還有人說「我月薪制沒加班費」——錯。
月薪一樣要折時薪算:月薪 ÷ 30 ÷ 8(施行細則 §11)。

存起來,下次對帳用得到。
```
**留言區(釘選):** 三種日自動套倍率、附條號 👉 [民眾版連結]
**法源:** 勞基法 §24、§39、施行細則 §11

### ⑤ 「長子多分、女兒沒份」是假的 〔破除迷思·引戰〕

```
「長子要多拿一份。」
「女兒嫁出去就沒繼承權。」

這兩句話在台灣家庭講了幾十年——
但民法從來沒這樣寫過。

民法 §1138、§1144 白紙黑字:
子女不分性別、排行、已婚未婚,應繼分一律均分。
配偶永遠是繼承人。

你家分財產時,有長輩講過這種話嗎?
```
**留言區(釘選):** 配偶搭不同順位、應繼分自動算 👉 [民眾版連結]
**法源:** 民法 §1138、§1144

### ⑥ 遺囑能不能「全部給一個人」? 〔知識反差〕

```
「我想把房子全留給最孝順的小女兒,其他人都不分。」

法律的答案是:部分可以,部分不行。
不能被剝奪的那塊,叫「特留分」(民法 §1223)。

直系血親、配偶的特留分 = 應繼分的 1/2。

舉例:爸爸留 1,200 萬,配偶+2 子女應繼分各 400 萬。
就算遺囑寫「800 萬全給小女兒」,
配偶和大女兒的特留分各 200 萬還是保得住。

所以「寫進遺囑就 100% 有效」是迷思。
```
**留言區(釘選):** 特留分一鍵試算 👉 [民眾版連結]
**法源:** 民法 §1223、§1225

### ⑦ 離婚時,老公賺的錢我真的沒份嗎? 〔憤怒共鳴〕

```
「公司是我開的、錢是我賺的、房子登記我名下——離婚你一毛拿不到。」

這句話在法律上站不住。

台灣預設「法定財產制」:婚後賺的,不管登記在誰名下,
都算「婚後財產」。離婚時用「剩餘財產分配請求權」(民法 §1030-1),
剩餘較少的一方可以要回差額的一半。

只有三種不列入:繼承來的、別人贈與的、慰撫金。

⚠️ 時效:知道差額起 2 年、或關係消滅起 5 年,過了就沒了。
```
**留言區(釘選):** 剩餘財產差額自動算 👉 [民眾版連結]
**法源:** 民法 §1030-1、§1030-3

### ⑧ 每年給小孩多少不用繳稅? 〔知識點〕

```
「每年包多少給小孩,不用報贈與稅?」

2022 年起:每位贈與人每年免稅額 244 萬(遺贈稅法 §22)。
重點來了——夫妻可各自用,
所以爸媽一年合贈子女最多 488 萬,免贈與稅。

超過怎麼課?(§19)
・淨額 ≤ 2,811 萬:10%
・~5,621 萬:15%
・再上去:20%

⚠️ 別玩「先轉太太帳戶再轉小孩」這招,
國稅局看資金流向,實質課稅照樣抓。
```
**留言區(釘選):** 跨年度分批贈與怎麼最省 👉 [民眾版連結]
**法源:** 遺贈稅法 §22、§19、§20

### ⑨ 爸留 3,000 萬房子,我要繳 300 萬遺產稅? 〔反直覺〕

```
「爸留給我 3,000 萬的房子,遺產稅 10% = 300 萬,對吧?」

錯。差很多。

遺產稅是課「淨額」不是「總額」。
先扣:免稅額 1,333 萬 + 喪葬費 138 萬 + 配偶 553 萬 + 子女每人 56 萬…

實算(配偶 1+子女 2):
扣除合計 = 1,333+138+553+56×2 = 2,136 萬
課稅淨額 = 3,000 − 2,136 = 864 萬
稅額 = 864 × 10% = 86.4 萬

不是 300 萬,是 86.4 萬。少算嚇自己,多算被人坑。
(以上為 2025 年起現行數字)
```
**留言區(釘選):** 帶入你家狀況自動算遺產稅 👉 [民眾版連結]
**法源:** 遺贈稅法 §13、§17、§18

### ⑩ 「保險金免稅」是真的嗎? 〔破除迷思〕

```
「買保險指定受益人,身故給付完全免稅」——這句話一半對一半錯。

對的部分:符合要件(死亡保險金 + 有指定受益人 + 人壽險),
保險金不計入遺產(保險法 §112)。

錯的部分:金額大時,還有第二道關卡 AMT(最低稅負制)。
2025 年起保險給付排除額 3,740 萬,
超過部分計入基本所得,扣 750 萬免稅額後 ×20%。

例:領 5,000 萬 →(5,000−3,740−750)×20% = 102 萬 AMT。

而且財政部還有「實質課稅」:高齡、臨終、躉繳、鉅額投保,
可能被認定假保險真避稅,照樣併入遺產。
```
**留言區(釘選):** 保險 AMT 一鍵試算 👉 [民眾版連結]
**法源:** 保險法 §112、所得基本稅額條例 §12
