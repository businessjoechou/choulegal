LIVE DEMO

我們花五分鐘，
用 30 行，做一個會呼叫 AI 的小東西

把一堆雜亂的使用者留言，自動分類、統計、畫成圖。
你會發現——它一點都不像魔法。

DEMO · 第一步：叫 AI 判斷一則
classify.py
import pandas as pd
from anthropic import Anthropic

ai = Anthropic()
df = pd.read_csv("留言.csv")        # 一欄 text，共 1000 則

def 分類(text):
    r = ai.messages.create(
        model="claude-haiku-4-5", max_tokens=10,
        messages=[{"role": "user",
            "content": f"只回 正面/負面/中立：\n{text}"}])
    return r.content[0].text.strip()

這 10 行就是整個 AI 的核心：把一段文字丟給模型，拿回一個判斷。看懂它，你就懂「程式怎麼呼叫 AI」了。

DEMO · 第二步：讓它跑完 1000 則，畫成圖
classify.py（續）
# 一行，套用到全部 1000 則 —— 這就是「自動化」
df["情緒"] = df["text"].apply(分類)

# 統計 + 一行畫圖
print(df["情緒"].value_counts())
df["情緒"].value_counts().plot.bar()

關鍵就是框起來那一行 .apply(分類)——「對每一則都做一次」。
人要做一千次的事，程式一行交代完。這，就是我說的槓桿。
