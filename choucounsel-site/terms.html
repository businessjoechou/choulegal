資料保存與使用界線

本頁說明 ChouCounsel 目前可用的案件資料保存、備份、保密、AI 草稿與本機使用界線。雲端長期保存、多人權限控管與自動跨裝置同步，需以帳號權限、資料庫服務與雙方約定的啟用範圍為準。

一、目前可用範圍

ChouCounsel 目前可整理案件、期限、任務、卷證、關係人、利益衝突線索、時數帳務、書狀模板、文件草稿、案件交接清單與工作台 JSON 備份。這些功能可作為律師或法律服務團隊的案件整理工具，但不代表法院、主管機關、客戶或第三方已接受任何結論。

目前案件資料保存在使用者自己的瀏覽器。更換裝置、清除瀏覽器資料、使用無痕模式或瀏覽器政策變更，可能導致資料無法保留；正式交接、存證或長期保存前，請匯出 JSON 備份並放入律所核准的保存位置。

二、案件資料與保密責任

使用者輸入的案件事實、當事人資料、卷證摘要、時數與文件草稿，應由使用者確認是否適合放入目前的本機工具。請勿在未確認授權、保密義務、個資法要求、律師倫理或律所內規前，輸入不應保存於試用或展示環境的敏感資料。

ChouCounsel 案件資料不得與民眾權益查詢資料混用。若律所或企業已有正式文件管理、客戶資料庫、稽核保存或資安流程，仍應以該正式流程作為主要保存位置。

三、備份與完整性碼

工作台可匯出 JSON 備份，備份會附完整性碼，匯入時會重新檢查，協助確認檔案內容未被改動；完整性碼只代表檔案內容一致，不代表案件事實正確、法律判斷正確、資料來源完整或已經人工複核。

交接清單與工作包是案件管理輔助輸出，適合提醒期限、待補卷證、待核對文件、利衝線索與帳務暫估；實際提交給客戶、法院、主管機關或對造前，仍需由承辦律師或授權人員確認。

四、AI 草稿與法律複核

ChouCounsel 的 AI 摘要、文件草稿、客戶更新稿、準備稿與模板內容，僅供整理與起草輔助，不構成律師法律意見、個案結論、訴訟策略保證或正式文件核定。所有法律引用、事實敘述、金額、期限與附件，均應回到卷證與官方來源核對。

五、雲端服務能力界線

本頁不代表雲端長期保存、多人權限控管或自動跨裝置同步已啟用。若日後啟用帳號權限、資料庫、成員角色、審計紀錄、跨裝置同步或商務條件，資料保存期限、刪除方式、權限設計、匯出限制、資安責任與服務終止條款，會另行公告或於雙方合約約定。
